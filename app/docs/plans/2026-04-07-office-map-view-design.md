# Office Map View — 설계 문서

> 2026-04-07 확정

## 개요

프로젝트 워크스페이스에서 대화 화면과 토글로 전환되는 전체 화면 오피스 맵 뷰. 추상 플로어플랜 스타일로 방을 색면 카드로 표현하고, 캣로프 아바타로 에이전트 배치를 보여준다.

## 확정된 결정 사항

| 항목 | 결정 |
|------|------|
| 맵 배치 | 전체 화면 맵 뷰 (토글 전환) |
| 방 클릭 동작 | 맵 유지 + 슬라이드 오버레이 (데스크탑), 바텀시트 (모바일) |
| 시각 스타일 | 추상 플로어플랜 — 타일 바닥 없이 색면 카드 + 아이콘 |
| 테마 색상 용도 | 채팅창 배경에 적용. 맵 카드에는 10-15% opacity로 힌트 |

## 테마 색상 매핑

| 방 | 맵 카드 배경 | 채팅 배경 |
|---|---|---|
| The Lobby | `#181412` at 10% | `#181412` |
| The Meeting Room | `#08090E` at 10% | `#08090E` |
| The Archive | `#F4F1EA` at 15% | `#F4F1EA` |
| The Individual Zones | `#FFF5F7` at 15% | `#FFF5F7` |

> 다크 톤은 10%, 라이트 톤은 15% — neutral 배경 위에서 비슷한 체감 강도

## 방 카드 디자인

- 라운드 카드 형태
- 배경: 해당 방의 테마 색상 10-15% opacity
- 방 이름 + 간단한 설명 텍스트
- 소속 에이전트 캣로프 아바타 (24px, 겹쳐서 표시)
- 상태 인디케이터: 새 메시지 뱃지, 대기 중 결정 카운트
- 대표(CEO) 캣로프는 현재 활성 방에만 표시

## 방 정보 슬라이드 (데스크탑 오른쪽 / 모바일 바텀시트)

- 방 이름 + 테마 컬러 헤더
- 참여 에이전트 목록 (캣로프 48px + 이름 + 역할)
- 최근 활동 3건
- 대기 중인 결정/액션 카운트
- "들어가기" 버튼 → 대화 화면으로 전환

## 대표 캐릭터 규칙

- 현재 활성 방(마지막으로 들어간 방)에 대표(CEO) 캣로프 표시
- 다른 방에는 표시 안 됨
- "대표가 지금 어디에 있는지" 한눈에 파악 가능

## 에셋 영향

- 타일셋 불필요 (추상 플로어플랜이므로)
- 기존 캣로프 캐릭터 south.png 재활용
- 추후 방 아이콘 필요할 수 있음 (v2)
