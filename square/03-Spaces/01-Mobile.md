# 01-Mobile

> 대표용 컴패니언 채널. 풀 기능 복제 앱이 아니라, 오피스 밖에서도 빠르게 확인하고 승인할 수 있는 외부 회선이다.

## 역할

- War Room 빠른 확인
- 에이전트 DM 짧은 답장
- 승인 / 보류
- 긴급 호출
- 짧은 메모 남기기

## 원칙

- 웹 오피스의 축소판이 아니라 대표의 포켓 인터컴으로 설계한다.
- 긴 문서 편집이나 복잡한 운영 작업은 웹에서 한다.
- 상태 확인과 짧은 개입이 가장 중요하다.

## 시각 방향

- floorplan 안에서는 방보다 `대표 전용 연결 노드`처럼 보이게 한다.
- 모바일 자체도 픽셀 전체 화면보다 현대적 UI 위에 작은 픽셀 포인트를 얹는 방식이 맞다.
