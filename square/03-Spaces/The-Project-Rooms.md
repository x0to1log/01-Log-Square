# The Project Rooms

> 각 프로젝트가 자기만의 맥락, 대화, 결정, 산출물을 가지는 작업실 묶음. 내부 구조는 `floorplan shell + messenger core + operations panel`이다.

## 역할

- 프로젝트별 대화 보관
- 진행 상태 추적
- 회의 요약과 문서 링크 연결
- 관련 Specialist 호출 기록 축적
- 프로젝트 전체를 오피스 도면 기반으로 탐색

## 원칙

- 하나의 프로젝트는 하나의 방을 가진다.
- 방 안에는 목표, 상태, 열린 질문, 최근 결정이 항상 보여야 한다.
- 프로젝트가 끝나면 요약 후 [[The-Archive]]로 이동한다.
- 프로젝트를 열면 기본으로 [[The-Boardroom|The War Room]]이 열린다.
- 각 에이전트와의 DM은 이 프로젝트 맥락 안에서만 이뤄진다.

## 기본 구조

- 상단: 프로젝트 헤더, phase, 열린 리스크, 대기 승인
- 좌측: War Room + 에이전트 DM 목록
- 중앙: 현재 대화
- 우측: 결정, 리스크, 액션, 관련 문서
- 보조 보기: 탑다운 픽셀 오피스 맵
