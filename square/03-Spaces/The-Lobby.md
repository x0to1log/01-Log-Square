# The Lobby

> 대표가 오피스에 들어와 가장 먼저 마주하는 홈. 기본 진입점은 `프로젝트 목록`이다.

## 역할

- 현재 프로젝트 목록 표시
- 오늘의 우선순위와 최근 활동 요약
- 대기 중인 승인과 열린 리스크 확인
- 각 프로젝트의 floorplan workspace 진입

## 보여줘야 하는 정보

- 현재 활성 프로젝트
- 최근 회의 3건
- 대기 중인 승인 항목
- QA의 열린 `Red Flag`
- 프로젝트별 상태, 최근 대화, 열린 액션 개수
